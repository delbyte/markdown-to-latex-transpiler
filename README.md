# Markdown to Latex Style PDF Transpiler

I have tested it here with my college submissions, can be used to write research papers in an easy Markdown format, and then convert it to professionally styled Latex PDFs. 

## Examples of generated PDFs can be found in the root of the repository.

# How to use -
1. Write a markdown file.
2. Run the ```converter.js``` with the Markdown file as an argument in your CLI.
